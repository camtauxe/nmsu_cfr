Course Funding Request Help

Welcome to the Course Funding Request system! As a submitter, you can submit course funding requests for your department as well as fill out your department's sources of salary savings. Use the navigation bar at the top of the page to navigate to different pages. Below is a brief description of each page and instructions on how to use them.

Course Funding Request

This page allows you to submit a course funding request or revise the current request. The table will show the current most recent version of your department's request. You can edit any of the values in the table or press the '+' button to add a new course to the request. To remove a course, check the checkbox in the last column. When you are done making your changes, press 'Submit Changes'.

Sources of Salary Savings

This page allows you to specify the sources of salary savings for your department. The table will show the current most recent version of your department's savings. You can edit the values in the table or press the '+' to add a new entry. To remove an entry, check the checkbox in the last column. When you are done making your changes, press 'Update'.

Current Semester Revisions

This page displays the history of revisions for your department's request this semester.

Previous Semester CFRs

This page displays the complete history of revisions for your department's requests for all previous semesters.
